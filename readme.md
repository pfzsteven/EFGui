# 简介

Python+Qt 制作效率工具，方便效果设计师在打开/导出 滤镜zip包时，可以立即检测内容是否合法等。大家可以修改python源码进行定制处理。

# 使用说明

该编辑器使用Python语言开发，打包成MacOS支持的`.app`文件，建议把该文件拖动到`访达(Finder)/应用程序` 当中，以后可从启动点直接点击运行.

## 功能介绍

### 双击

- 图片类型。打开新窗口预览该图片。
- 文本类型。进入文本编辑界面。
- 文件夹类型。展开或者收起。

### 右键点击

- 文件夹类型。(1)新建子文件或子文件夹; (2)删除
- 文件类型。(1)编辑;(2)删除
